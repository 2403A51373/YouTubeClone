# YouTube Clone (HTML & CSS)
A front-end clone of the YouTube homepage built using HTML and CSS.  
This project focuses on layout design, responsive UI, and modern CSS styling.

## Features
- YouTube-like homepage layout
- Responsive video grid
- Navigation sidebar
- Header with search bar
- Video thumbnails with channel details

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)

## Purpose
This project was created to practice front-end development and understand how large video platforms structure their UI.

## Future Improvements
- Add JavaScript functionality
- Implement responsive mobile design
- Add video playback page
